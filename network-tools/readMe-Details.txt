For probing computer networks, Nmap offers a variety of functionalities, including as host discovery, service detection, and operating system detection. Scripts that offer more sophisticated service discovery, vulnerability detection, and other features can extend these features. During a scan, Nmap can adjust to changing network conditions, such as latency and congestion. 
Nmap was originally a Linux tool, but it has now been made available for Windows, macOS, and BSD. Following Linux in popularity is Windows.
